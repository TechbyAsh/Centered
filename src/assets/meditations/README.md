# Meditation Audio Files

This directory contains meditation audio files for the guided mini-meditations feature.

## Current Files
- `placeholder.mp3`: A placeholder file for development. In production, this should be replaced with actual guided meditation audio files:
  - Quick Calm (3 minutes)
  - Mindful Break (5 minutes)
  - Stress Relief (7 minutes)

## Audio File Requirements
- Format: MP3
- Quality: 128-192 kbps
- Sample Rate: 44.1 kHz
- Channels: Stereo
- Duration: As specified per meditation

## Production Notes
Before deploying to production:
1. Replace placeholder.mp3 with actual meditation audio files
2. Ensure audio quality meets requirements
